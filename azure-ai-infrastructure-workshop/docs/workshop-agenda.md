# Workshop agenda

## Standard one-day path

- Welcome, architecture, platform boundaries
- Preflight and quota validation
- Shared AKS foundation
- KAITO inference lab
- KAITO fine-tuning guided lab
- Anyscale on Azure orientation
- Ray Data batch job
- Ray Serve service
- Ray Train guided exercise
- Foundry managed compute (dedicated GPU endpoint)
- Fireworks AI on Foundry (partner Models-as-a-Service)
- Observability, comparison, and cleanup

Recommended balance: approximately 30% explanation and 70% hands-on activity. High-cost, multi-GPU examples should be instructor demonstrations unless quota and budget are confirmed.
