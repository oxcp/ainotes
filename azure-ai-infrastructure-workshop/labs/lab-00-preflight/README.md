# Lab 00: Preflight

## Objectives
- Validate local tools, Azure access, provider registration, region, and quota.
- Confirm Anyscale on Azure access without trying to install an operator.

## Prerequisites
An Azure subscription and instructor-provided Anyscale on Azure access.

## Exercises
1. Copy `.env.sample` to `.env`.
2. Run `../../scripts/check-prerequisites.sh`.
3. Run `../../scripts/check-quota.sh`.
4. Sign in to the instructor-provided Anyscale on Azure environment and confirm the assigned cloud and project are visible.
5. Record blockers before provisioning.

## Success criteria
Azure context is correct, quota is reviewed, and assigned Anyscale cloud/project access is confirmed.

## Cleanup
No resources are created.
