# Lab 05: Ray Data batch processing

Use the provisioned Anyscale on Azure Workspace or Job workflow.

## Steps

1. Upload `data/sample-prompts.jsonl` with the lab files.
2. Run `src/batch_processing.py` first in low-cost CPU mode.
3. Inspect row count, schema, task parallelism, and output.
4. Optional instructor track: replace the sample transform with a validated model predictor on approved GPU compute.

The default exercise demonstrates Ray Data mechanics. It does not claim to run model inference.
